# exoplanet-light-curve-analaysis
## Lightcurves from telescopes can be used to identify transitting exoplanets.
### What is a lightcurve?
#### Lightcurves are data points from a telescope that measures the brightness (magnitude or a flux) over time. 
## Typically, when an exoplanet transits in front of a star, there is a dip in brightness.
## Kepler telescope data is useful for this
## With this project, I intend to identify possible exoplanet transits from light curves.
